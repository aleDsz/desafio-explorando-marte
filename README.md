# Desafio: Explorando Marte

## Running

Just run `elixir app.exs`

## How to use?

- Set inputs separated by space
- You can send these type of data:
  - `ACTION`
  - `X Y`
  - `X Y ACTION`
  - `X Y DIRECTION`
  - `X Y DIRECTION ACTION`

## Data types

- **DIRECTION**: Initial direction (N/E/S/W)
- **ACTION**: One or more actions (M/L/R)
- **X**: Coordenate x
- **Y**: Coordenate y
